# CalorieTrack

A private mobile-style calorie and weight tracker web app.

## Features

- Manual food entry
- Daily calorie total
- Protein, carbs, and fat tracking
- Saved foods for quick add
- Weight tracking
- Estimated BMR and maintenance calories using Mifflin-St Jeor
- Goal target for maintain, cut, bulk, or custom calories
- Export/import backup
- PWA support for adding to iPhone Home Screen

## GitHub Pages upload

Upload these files directly into the repository root:

- `index.html`
- `styles.css`
- `app.js`
- `manifest.json`
- `sw.js`
- `icon-180.png`
- `icon-192.png`
- `icon-512.png`
- `README.md`

Then enable GitHub Pages from Settings → Pages → Deploy from a branch → `main` → `/ root`.

## iPhone install

Open the GitHub Pages link in Safari, tap Share, then tap Add to Home Screen.

## Data note

The app stores data locally in the browser. Export backups occasionally.
